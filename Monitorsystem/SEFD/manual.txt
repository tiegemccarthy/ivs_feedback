SEFD

To re-estimate SEFD values and plot vs scheduled SEFD, transfer:
•	files <SessionName>.corr.perl into folder \SEFD\corrperl
•	filse <SessionName>.skd into folder \SEFD\sked

Run GO_SEFD and all plots are stored in the output folder \SEFD\output.

main file: GO_SEFD
data: <SessionName>.corr.perl, <SessionName>.skd
output: plot of re-estimated SEFD values and scheduled SEFD values of X- and S-Band for each station per session
