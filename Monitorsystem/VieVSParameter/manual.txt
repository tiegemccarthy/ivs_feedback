VieVSParameter

Results of the VieVS adjustment are stored in …\VieVS\DATA\LEVEL3. This data is used to estimate and plot following parameters:
a.)	Number of scheduled vs real observations
b.)	Standard deviation of unit weight
c.)	Errors of station coordinates
Using the VieVS folder structur it is only necessary to set the path to VieVS. Additionally set the station names and the processlist that was used to load the VieVS input. 

Run GO_VieVSParameter and all plots are stored in the output folder \VieVSParameter\output.

main file: GO_VieVSParameter
data: …\VieVS\DATA\LEVEL3
output: plot of Number of scheduled vs real observations + trend, Standard deviation of unit weight,Errors of station coordinates
